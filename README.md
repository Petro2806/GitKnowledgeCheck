# Формат контрольної роботи

## 1) Отримання завдання
Індивідуальне завдання для кожного студента буде додано у папку `task` у вигляді текстового файлу з вашим прізвищем.

## 2) Час на виконання
На виконання завдання відводиться приблизно **5 хвилин**. За базового знання стандартних команд Git більшість завдань можна виконати за **1–2 хвилини**.

Такий таймінг потрібен, щоб я мав(ла) змогу одночасно контролювати виконання **двох студентів**. Якщо комусь буде потрібно трохи більше часу — за потреби час може бути продовжено. Водночас час обмежений, щоб протягом однієї пари всі мали можливість продемонструвати свої знання.

## 3) Що здати після виконання
Після завершення завдання виконайте в терміналі такі команди:

```bash
git status
git remote -v
git log --oneline --graph --decorate -n 15
```

Далі у папці [results](./results) створіть текстовий файл з вашим прізвищем і вставте туди вивід цих команд.
